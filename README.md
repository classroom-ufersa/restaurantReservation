# Restaurant Reservation

## Basic attributes of restaurant:
- Location;
- Opening time;
- Closing time;
- Total tables;
- Total bookings;
- Reservations;

## Reservation content: 
- Name of person in charge;
- Start time;
- Departure time;
- Table number;
- Consumption;

## Menu options:
1. Add Reservation;
2. Delete reservation;
3. List marked reservations;
4. Search for reservation;
5. Edit reservation record;
6. Consult reservation record;
7. Exit;
